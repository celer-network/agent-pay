# Celer AgentPay State Channel Network

Official go/grpc implementation of the off-chain modules of Celer AgentPay state channel network.

Celer AgentPay state channel network is a generic framework of state channels with deeply optimized on-chain contracts and off-chain messaging protocols. This repo implements the off-chain nodes.
